# List of Importand Cloud Foundry Commands

`cf orgs`  
Lists all organizations.

`cf spaces`  
Lists all spaces  
in the current organization.

`cf target`  
Shows the current target.

`cf target -o <org>`  
Switches the organization.

`cf target -s <space>`  
Switches the space.

`cf target -o <org> -s <space>`  
Switches both.

`cf apps`  
Lists all apps.

`cf services`  
Lists all services.

`cf help`  
Shows the help page.

`cf login [--sso]`.
Login to Cloud Foundry. Use cf login --help for more Information.
 
`cf deploy <mtar-file>`
Deploy Project
